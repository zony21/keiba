# 2026年 七夕賞 再テスト

## 目的
GitHubの最新採点ロジック v2.1 を使用し、七夕賞の全出走馬を3段階で再採点する。

## 今回の条件
- ネット検索は使用しない。
- ユーザー提供データのみを使用する。
- 未取得情報は推測せず、空欄または未取得として記録する。
- 全頭の採点が完了するまで、印・買い目・期待値順位を出さない。
- オッズは採点完了後にのみ使用する。

## 保存ファイル
- `score.csv`：全頭の項目別点数と最終点
- `score_detail.md`：各馬の点数の理由、加点理由、減点理由、未取得情報
- `prediction.md`：全頭採点完了後の印、期待値、買い目
- `review.md`：確定結果と採点検証

## 現在の進行状況
第1段階「過去実績」の採点準備中。
