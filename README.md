# keiba

競馬予想を全頭3段階で採点し、結果を蓄積するリポジトリ。

## 正式ロジック

- `docs/scoring_logic_v2.md`
- `templates/race_score.csv`

旧ロジックは使用しない。

## 実行手順

1. 正式ロジックを読む
2. 全頭の第1段階を採点する
3. 全頭の第2段階を採点する
4. 当日情報があれば第3段階を採点する
5. 合計順位を確定する
6. オッズから期待値と買い方を決める
7. 採点表と予想を保存する
8. 結果後に検証を追記する

全頭採点が終わるまで印を付けない。未取得情報は推測しない。不的中は0点とする。
