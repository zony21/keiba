# keiba

競馬予想を、長文ロジックではなく**3段階の採点方式**で管理するためのリポジトリです。

## 方針

旧「競馬AI予想 判断ロジック Ver.6.0」の評価ロジックは破棄します。

残すものは次の3つだけです。

1. 情報取得の確認項目
2. 馬券予想の出力形式
3. 結果検証の記録項目

今後の予想は、全出走馬について3段階で採点し、最終合計点が高い馬を中心に着順候補と買い目を作成します。

## 3段階

- 第1段階：基礎能力・過去実績
- 第2段階：今回条件への適合
- 第3段階：当日状態・市場評価

年齢、近走、過去実績、条件不一致などのマイナス材料は、明示的に減点します。

## ファイル

- `docs/scoring_logic_v1.md`：現行の採点ロジック
- `templates/race_score.csv`：全頭採点テンプレート
- `records/README.md`：レースごとの記録方法

## 運用ルール

- 予想ごとに必ずこのリポジトリの現行ロジックを確認する
- 全頭を同じ項目で採点する
- 未取得項目を勝手に0点扱いしない
- 未取得と評価済みを区別する
- オッズだけで順位を決めない
- オッズは期待値と馬券配分に反映する
- レース終了後に採点と結果を保存する
