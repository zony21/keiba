# 競馬AI採点ロジック v2.0

## 0. 目的

全情報を並べるのではなく、**予想順位を変える情報だけを採点する**。

情報量を増やすことを目的にしない。
全頭を同じ項目・同じ尺度で採点し、3段階の合計点が高い馬を中心に予想する。

旧ロジックの長文説明は予想時に使用しない。
過去に収集した情報は破棄せず、各採点項目の根拠としてだけ利用する。

---

# 1. 予想前の最低確認

次だけを確認する。

- レース：開催日、競馬場、レース番号、距離、芝・ダート、クラス
- 出走馬：馬番、馬名、年齢、斤量、騎手
- 過去成績：今回に近いレース、近走内容、着差、通過順、相手レベル
- 今回条件：コース、距離、馬場、展開、枠
- 当日状態：馬体重、調教、パドック（取得できる範囲）
- オッズ：採点後の期待値・買い方にのみ使用

取得しても採点に使わない情報は、予想本文へ並べない。

---

# 2. 第1段階：過去実績点

目的：過去レースから、現在の基礎能力を評価する。

最大30点。

## 2.1 今回に近い条件の実績：0〜10点

今回と近い順に評価する。

- 同競馬場・同距離・同馬場・同クラス
- 同じコース形状・同距離
- 同距離・近い相手レベル

単なる着順ではなく、着差・通過順・展開・斤量も見る。

## 2.2 相手レベルとレース内容：0〜8点

- 強い相手に小差：高評価
- 弱い相手に完敗：低評価
- 展開不利や明確な不利があった場合は補正

## 2.3 近走の安定性・上昇度：-6〜6点

- 近走内容が安定、または改善：加点
- 大敗連続、着差拡大、位置取り悪化：減点

## 2.4 勝ち切る力：0〜6点

- 自分から動いて勝てる
- 展開に依存せず好走できる
- 同条件で勝ち切った実績がある

## 2.5 年齢・能力低下減点：0〜-4点

年齢だけでは減点しない。
次が見られる場合のみ減点する。

- 近走着差の拡大
- 位置取りの悪化
- 反応・上がりの低下
- 過去実績を現在再現できていない

---

# 3. 第2段階：今回条件点

目的：基礎能力を今回発揮できるか評価する。

最大30点。

## 3.1 コース・距離・馬場適性：0〜10点

- 直接実績を最優先
- 類似コース実績を次に評価
- 血統は直接実績が少ない場合の補助に限定

## 3.2 展開・想定4角位置：-6〜8点

- 今回のペースで理想位置を取れる
- 小回りなら4角で勝負圏に入れる
- 逃げ競合、後方固定、展開依存は減点

## 3.3 斤量：-4〜4点

- 過去に同斤量以上で好走：減点しない
- 前走から有利な減量：加点
- 重い斤量で内容悪化が続く：減点

## 3.4 枠順：-3〜3点

コース形状・脚質との組み合わせで評価する。

## 3.5 騎手：-3〜3点

次だけを見る。

- この馬との実績
- 今回コースでの位置取り・判断
- 継続騎乗または明確な強化・弱化

騎手名の知名度では加点しない。

## 3.6 血統：-2〜2点

父・母・母父の情報は、今回条件への補助根拠としてのみ使う。
直接実績を血統で逆転させない。

---

# 4. 第3段階：当日状態点

目的：能力と適性を当日発揮できるか確認する。

最大10点。

## 4.1 馬体重・推移：-4〜4点

- 好走時体重との一致
- 連続減少、輸送減り、太め残り
- 成長分、絞れ、疲労

増減の数字だけで採点しない。

## 4.2 パドック：-3〜3点

- 歩様
- 発汗
- テンション
- 馬体の張り

未取得なら空欄。0点扱いしない。

## 4.3 調教：-2〜2点

- 能力順位を作らない
- 明確な良化・悪化だけ反映
- A・Bなどの記号だけで加点しない

## 4.4 当日馬場補正：-1〜1点

事前想定と実際の馬場が違う場合のみ微調整する。

---

# 5. 合計と順位

最終点：

`第1段階 + 第2段階 + 第3段階`

原則として合計点順に能力順位を作る。

同点または2点以内の場合だけ、次で比較する。

1. 今回に近い条件の実績
2. 展開・4角位置
3. 斤量
4. 当日状態

採点後に、1着・2着・3着の性質を分ける。

- 1着候補：勝ち切る力と自力で動ける能力
- 2着候補：安定性と崩れにくさ
- 3着候補：条件特化、展開利、軽斤量、一芸

---

# 6. オッズの扱い

オッズは能力点へ入れない。
ただし無視もしない。

採点完了後に次を確認する。

- 点数上位なのに人気がない理由
- その不人気理由が今回も有効か
- 点数に対して過剰人気か
- 点数に対して過小評価か

オッズは次へ反映する。

- 期待値順位
- 券種
- 買い目
- 資金配分
- 見送り

---

# 7. 省略防止ルール

- 全頭の3段階採点が終わるまで印を付けない
- 一部の馬だけ詳細に採点しない
- 未取得は空欄とし、推測しない
- 根拠が同じ情報を複数項目で重複加点しない
- 予想本文には上位馬と危険馬の根拠だけ記載する
- 全頭の詳細は採点表へ保存し、本文を長文化しない

---

# 8. 結果検証

保存するものは次だけとする。

- 確定着順
- 事前合計点と順位
- 買い目、投資、払戻、回収率
- 1〜3着馬の過小評価項目
- 上位予想馬の過大評価項目
- 取得漏れ
- 次回変更する配点

不的中は0点。
結果後に事前採点を書き換えない。
