# レース記録

各レースの採点・予想・結果検証を、このフォルダへ保存する。

## 保存単位

1レースにつき、最低でも次の2ファイルを作る。

- `YYYY-MM-DD_競馬場R_レース名_score.csv`
- `YYYY-MM-DD_競馬場R_レース名_review.md`

## score.csv

`templates/race_score.csv` をコピーし、全出走馬を1行ずつ記録する。

必須事項：

- 全頭を記録する
- 未取得項目は空欄にし、`missing_info`へ理由を書く
- 予想確定後に能力点を都合よく変更しない
- 結果判明後は結果列と検証列のみ追記する

## review.md

以下を記録する。

1. 対象レース
2. 情報取得状況
3. 第1段階順位
4. 第2段階順位
5. 第3段階順位
6. 最終合計順位
7. 1着・2着・3着候補
8. オッズと期待値判断
9. 買い目と資金配分
10. 確定結果
11. 回収率
12. 過大評価馬
13. 過小評価馬
14. 取得漏れ
15. 採点ミス
16. 重み付けミス
17. 次回改善案

## 更新禁止事項

- 外れた後に事前点数を書き換える
- 予想根拠を結果に合わせて変更する
- 的中していない馬券を「実質的中」と扱う
- 「惜しい」「軸は来た」などで加点する

不的中は0点として残す。
